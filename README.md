- 基于[Nonebot2](https://github.com/nonebot/nonebot2)，有良好的生态支持
- 开箱即用，无需复杂配置
- 集成包管理器，支持一键安装插件
- 支持 OneBot 标准通信但不限于此
- 自定义主题支持，满足审美需求
- 国际化支持，支持多语言本地化
- 高性能，500 插件 10s 内启动
